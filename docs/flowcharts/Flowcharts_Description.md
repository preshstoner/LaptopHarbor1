Process Diagrams
Each module’s workflow explained:
    • Login Flowchart(login.png)
        Start → Enter Credentials → Validate → Success → Dashboard
        If invalid → Error Message → Retry

    • Shopping Cart Flowchart(Cart.png)
        Add Item → Update Cart → View Cart → Checkout Decision → Payment → Confirmation

    • Order Tracking Flowchart(OrderTracking.png)
        User Request → Fetch Order Status → Display Status → Notify Updates

    • Authentication Module(Authentication.png)
        Input: Username/Password → Process: Verify credentials → Output: Access granted/denied.

    • Product Module(Product.png)
        Input: Product query → Process: Search/filter database → Output: Product list.

    • Order Module(Order.png)
        Input: Cart items → Process: Payment + confirmation → Output: Order ID + tracking.
